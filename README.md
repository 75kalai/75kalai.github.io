# Infinity-Buildiers
This project is a static website done just to showcase what is possible ONLY using HTML and CSS. The project themed on architecture &amp; construction company's landing site done for an imaginary company called "infinity builders".

The animated building pages does not contain any images/videos or sprite files. It is all achieved using CSS rules.
The project does not resemble a real landing site. It's just a creative work showcasing CSS possibilities.
No CSS pre-processors or JS is used.
