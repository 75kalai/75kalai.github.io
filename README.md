# Infinity-Buildiers
This project is a static website done just to showcase what is possible ONLY using HTML and CSS. The project themed on architecture &amp; construction company's landing site done for an imaginary company called "infinity builders" as a story. 

The "animated building" pages(pages 8 & 9) does NOT contain any images/videos or sprite files. It is all achieved using CSS rules. Page 4 uses video as background, the cube is css.

The project does not resemble a real landing site. It's just a creative work showcasing CSS possibilities.
No CSS pre-processors or JS is used. 

Dont forget to check out till end.




> about the code
The "style.css" file contains the style code for the animatied buildings. This is written in a generic way so creating a new building will become easier. 
For example, just typing in HTML, .beam>(.front+.back+.left+.right) will generate a column/beam for the building.
